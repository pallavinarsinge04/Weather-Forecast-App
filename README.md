# Weather Forecast Web Application

A responsive web application that displays real-time weather data using OpenWeatherMap API.

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## Features
- Search weather by city
- Live temperature
- Weather condition
- Humidity and wind speed
- Error handling
- Responsive UI

## How to Run
1. Clone the repository
2. Add your API key in config.js
3. Open index.html in browser
